# Pong2D

Juego 2D en Unity inspirado en el clásico Pong, desarrollado como proyecto de clase.

## Objetivo
Crear una versión funcional del juego Pong con:
- Control de jugador
- Movimiento automático de la CPU
- Sistema de puntuación
- Reinicio de partida
- Interfaz con Canvas

## Tecnologías
- Unity
- C#
